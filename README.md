
# Core Data Structures & Conditional Logic in Python

## 📌 Overview
This repository demonstrates fundamental **Python data structures** (List, Dictionary, Set) and **Conditional Statements** (if, elif, else).  
It covers creation, modification, and access operations with practical examples.

---

## 🗂 Contents

### 1. Lists
- **Creation**: `age_list`, `name_list`
- **Operations**: append, insert, remove, pop, extend, sort
- **Aggregations**: max, min, sum
- **Accessing Elements**: indexing, slicing, reverse order

### 2. Dictionaries
- **Creation**: `student_marks`
- **Access**: retrieve values by key
- **Modification**: add, update
- **Inspection**: keys, values, items

### 3. Sets
- **Creation**: unique elements only
- **Properties**: no duplicates, no indexing
- **Operations**: union, intersection

### 4. Conditional Statements
- **User Input**: `score`
- **Logic**: if, elif, else
- **Output**: performance category based on score
